typescript+node

Para mostrar la consola en Chrome pulsa ctrl + shift + j

Por primera vez, cargar los modulos javascript, entre otros mocha y chai, ejecuar "npm install". Se cargarán estas librerias en el directorio "node_modules".

Para compilación de typescript ejecuta menu terminal --> Run Build Task... (Ctrl + Shift + B)

Si no se ejecuta correctamente, para realizar la compilación de typescript, ejecutar desde la terminal: tsc -b src test

Para depuración, abre index.html con Live Server y a continuación inicia la depuración.

Para ejecutar los tests con Mocha, abrir test.html

Para ejecutar la consola consola.html
